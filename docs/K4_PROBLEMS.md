# Grade 4 Math Problem Templates

## 1. Properties of Addition and Multiplication

### 1.1 Properties of Addition

#### Commutative Property

**Easy:**
```
{{name}} says that {a:10-99} + {b:10-99} is the same as {{b}} + {{a}}.
Is {{name}} correct? Explain why using the commutative property.
```

**Medium:**
```
Fill in the blank using the commutative property of addition:
{a:100-999} + {b:100-999} = _____ + {{a}}
```

**Hard:**
```
{{name_1}} has {a:50-200} [marbles | stickers | cards]. {{name_2}} has {b:50-200} [marbles | stickers | cards].
{{name_1}} says: "Together we have {{a}} + {{b}} [marbles | stickers | cards]."
{{name_2}} says: "Together we have {{b}} + {{a}} [marbles | stickers | cards]."
Who is correct? Use the commutative property to explain your answer.
```

#### Associative Property

**Easy:**
```
Calculate using the associative property:
({a:5-20} + {b:5-20}) + {c:5-20} = {{a}} + ({{b}} + {{c}})
Show that both sides are equal.
```

**Medium:**
```
{{name}} needs to add {a:25-75} + {b:25-75} + {c:25-75}.
Regroup the numbers to make the addition easier, then find the sum.
<!-- Hint: Choose b and c so they make a round number -->
```

**Hard:**
```
A [store | library | school] received donations over three days:
- Monday: {a:100-300} items
- Tuesday: {b:100-300} items  
- Wednesday: {c:100-300} items
Use the associative property to find the total in the easiest way possible.
Show two different groupings and prove they give the same answer.
```

#### Identity Property

**Easy:**
```
What number do you add to {a:10-999} to get {{a}}?
This is called the _____ property of addition.
```

**Medium:**
```
{{name}} had {a:50-500} [coins | stamps | books]. [He | She] received {b:0} more.
How many [coins | stamps | books] does {{name}} have now?
What property of addition does this show?
```

### 1.2 Properties of Multiplication

#### Commutative Property

**Easy:**
```
Show that {a:2-12} × {b:2-12} = {{b}} × {{a}}
```

**Medium:**
```
A rectangle has {a:3-15} rows of [tiles | squares | dots] with {b:3-15} in each row.
Another rectangle has {{b}} rows with {{a}} in each row.
Do both rectangles have the same total? Explain using the commutative property.
```

**Hard:**
```
{{name}} arranges {a:4-12} groups of {b:4-12} [chairs | desks | plants] in a room.
{{name_2}} rearranges them into {{b}} groups of {{a}}.
Without counting, explain why both arrangements have the same total number.
```

#### Associative Property

**Easy:**
```
Calculate: ({a:2-5} × {b:2-5}) × {c:2-5}
Then calculate: {{a}} × ({{b}} × {{c}})
Are the results the same?
```

**Medium:**
```
Find the product using the associative property to make it easier:
{a:2-5} × {b:4-8} × {c:5-10}
<!-- Choose values where two factors multiply to a round number -->
```

**Hard:**
```
A warehouse has {a:2-4} sections. Each section has {b:3-6} shelves. 
Each shelf holds {c:5-10} boxes.
Use the associative property to calculate the total boxes in two different ways.
```

#### Distributive Property

**Easy:**
```
Use the distributive property to solve:
{a:2-9} × {b:11-19} = {{a}} × ({{b_tens}} + {{b_ones}})
<!-- b_tens = tens digit of b × 10, b_ones = ones digit of b -->
```

**Medium:**
```
{{name}} wants to multiply {a:3-9} × {b:23-47}.
Break apart {{b}} to make the multiplication easier using the distributive property.
Show your work.
```

**Hard:**
```
A [theater | stadium | auditorium] has {a:6-9} sections.
Each section has {b:30-60} seats in the lower level and {c:20-40} seats in the upper level.
Use the distributive property to find the total number of seats.
Write the equation and solve.
```

#### Identity and Zero Properties

**Easy:**
```
Solve:
a) {a:15-999} × 1 = _____
b) {b:15-999} × 0 = _____
Name the property used in each.
```

**Medium:**
```
{{name}} says any number multiplied by {a:0-1} equals [the number itself | zero].
Is {{name}} correct? Give three examples to support your answer.
```

```

```
Which of these numbers are divisible by {d:3|4|6|9}?
{{n1}}, {{n2}}, {{n3}}, {{n4}}
<!-- Generate mix of divisible and non-divisible numbers -->

```

**Hard:**
```
Find a {digit_count:3|4}-digit number that is divisible by BOTH {d1:2|3|4|5} AND {d2:3|4|5|6}.
Explain how you checked using divisibility rules.
```

### 3.2 Easy Division Tricks

**Easy:**
```
Use a division trick to divide {n:20-100} by {d:2|5|10} mentally.

```

**Medium:**

A number has digits {d1:1-9}, {d2:0-9}, and {d3:0-9}.

### 3.3 Remainder Rules

**Easy:**

**Medium:**
```

```

**Hard:**
```

## 4. Operations on Fractions

### 4.1 Addition of Fractions

**Easy (Same Denominator):**
```
Add: {a:1-5}/{d:6-12} + {b:1-5}/{{d}}
<!-- Ensure a + b < d for proper fraction result, or allow improper -->
```

**Medium (Different Denominators):**
```
{{name}} ate {a:1-3}/{d1:4|6|8} of a pizza for lunch and {b:1-3}/{d2:2|3|4} for dinner.
How much pizza did {{name}} eat in total?
<!-- Choose d1, d2 where one is a multiple of the other or they share LCD -->
```

**Hard:**
```
Add and simplify:
{a:2-5}/{d1:6-12} + {b:1-4}/{d2:4-10} + {c:1-3}/{d3:3-8}
<!-- Ensure denominators have a reasonable LCD -->
```

### 4.2 Subtraction of Fractions

**Easy (Same Denominator):**
```
Subtract: {a:4-8}/{d:9-12} - {b:1-3}/{{d}}
<!-- Ensure a > b -->
```

**Medium (Different Denominators):**
```
{{name}} had {a:3-7}/{d1:8|10|12} of a [ribbon | rope | string].
[He | She] used {b:1-3}/{d2:2|4|5} for a project.
How much [ribbon | rope | string] is left?
<!-- Ensure a/d1 > b/d2 -->
```

**Hard:**
```
A tank was {a_whole:1-2} {a_num:1-3}/{a_den:4-8} full of water.
After using {b_num:1-4}/{b_den:3-6} of the tank, how much water remains?
Express your answer as a mixed number in simplest form.
```

### 4.3 Multiplication of Fractions

**Easy:**
```
Multiply: {a:1-4}/{b:5-10} × {c:2-6}
```

**Medium:**
```
{{name}} has {a:1-3}/{b:4-8} of a [chocolate bar | cake | pie].
[He | She] gives {c:1-2}/{d:3-4} of [his | her] piece to a friend.
What fraction of the whole [chocolate bar | cake | pie] did the friend get?
```

**Hard:**
```
A recipe calls for {a_whole:1-2} {a_num:1-3}/{a_den:4-8} cups of flour.
{{name}} wants to make {b:1-2}/{c:2-4} of the recipe.
How much flour does {{name}} need?
```

### 4.4 Division of Fractions

**Easy:**
```
Divide: {a:2-6}/{b:7-12} ÷ {c:2-4}
```

**Medium:**
```
{{name}} has {a:3-7}/{b:8-12} [meters | yards | feet] of ribbon.
[He | She] cuts it into pieces that are each {c:1-2}/{d:4-8} [meters | yards | feet] long.
How many pieces can {{name}} make?
```

**Hard:**
```
If {a:2-5}/{b:6-10} of a number equals {c:1-4}/{d:5-8}, what is the number?
Show your work using fraction division.
```

## 5. Decimal Numbers

### 5.1 Understanding Decimals

**Easy:**
```
Write the decimal for: {a:1-9} tenths
Write the decimal for: {b:1-9} hundredths
```

**Medium:**
```
{{name}} has {dollars:1-20} dollars, {dimes:1-9} dimes, and {pennies:1-9} pennies.
Write the total amount as a decimal.
```

**Hard:**
```
Arrange these decimals from least to greatest:
{a:1-9}.{b:0-9}{c:0-9}, {{a}}.{d:0-9}, {{a}}.{e:0-9}{f:0-9}{g:0-9}
<!-- Use same whole number to focus on decimal comparison -->
```

### 5.2 Place Value

**Easy:**
```
In the number {whole:1-99}.{tenth:0-9}{hundredth:0-9}, what digit is in the [tenths | hundredths] place?
```

**Medium:**
```
Write the value of the underlined digit:
{a:1-9}{b:0-9}.{c:0-9}{d:0-9}
<!-- Specify which digit is underlined in template -->
The {{position}} digit {{underlined_digit}} represents _____.
```

**Hard:**
```
{{name}} writes a decimal number. The digit {d:1-9} is in the tens place.
The digit in the tenths place is {t:1-9}. The digit in the ones place is {o:0-9}.
The digit in the hundredths place is twice the digit in the tenths place.
What is {{name}}'s number?
<!-- Ensure 2×t ≤ 9 -->
```

### 5.3 Adding and Subtracting Decimals

**Easy:**
```
Add: {a:1-9}.{b:1-9} + {c:1-9}.{d:1-9}
```

**Medium:**
```
{{name}} bought a [book | toy | snack] for ${a:5-20}.{b:00-99} and a [pencil | drink | sticker] for ${c:1-9}.{d:00-99}.
How much did {{name}} spend in total?
How much change from ${bill:20|50} will {{name}} receive?
<!-- Ensure total < bill -->
```

**Hard:**
```
{{name_1}} ran {a:10-15}.{b:0-9}{c:0-9} kilometers on Monday.
{{name_2}} ran {d:1-3}.{e:0-9}{f:0-9} kilometers more than {{name_1}}.
{{name_3}} ran {g:0-2}.{h:0-9}{i:0-9} kilometers less than {{name_2}}.
How far did {{name_3}} run?
```

## 7. Triangles

### 7.1 Types of Triangles by Sides

**Easy:**
```
A triangle has sides measuring {{a}} cm, {{b}} cm, and {{c}} cm.
What type of triangle is it?
<!-- For equilateral: a = b = c -->
<!-- For isosceles: exactly two sides equal -->
<!-- For scalene: all different -->
```

**Medium:**
```
{{name}} draws a triangle with two sides of {a:3-10} cm each.
The third side is {b:2-12} cm.
a) What type of triangle is this based on its sides?
b) What is the perimeter?
<!-- Ensure triangle inequality: a + a > b and a + b > a -->
```

**Hard:**
```
An isosceles triangle has a perimeter of {p:18-36} cm.
The two equal sides are each {a:6-12} cm long.
What is the length of the third side?
Is this a valid triangle? Explain.
<!-- Third side = p - 2a; check triangle inequality -->
```

### 7.2 Types of Triangles by Angles

**Easy:**
```
A triangle has angles measuring {a:30-60}°, {b:30-60}°, and {{c}}°.
What type of triangle is it based on its angles?
<!-- c = 180 - a - b -->
```

**Medium:**
```
{{name}} draws a triangle with one angle of {a:91-150} degrees.
a) What type of triangle is this?
b) Can this triangle have another angle greater than {threshold:89|90} degrees? Explain.
```

**Hard:**
```
In a right triangle, one of the acute angles is {a:15-75} degrees.
a) What is the measure of the other acute angle?
b) If the longest side (hypotenuse) is {c:10-20} cm, can you determine if this is scalene or isosceles?
<!-- Other acute angle = 90 - a; isosceles only if a = 45 -->
```
